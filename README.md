### About This Repository

This repo live on Render.

#### Tech Stack

- **Next.js & React** for the front end
- **Tailwind CSS** for styling
- **Node.js**, **TypeScript**, **Python** for back‑end services
- **Render** for hosting and CI/CD
- **GitHub Actions** for automated testing and deployments

#### Contributing

1. Clone the repository and install dependencies with `npm install`.
2. Push updates to trigger automatic deployments via Render.
3. For project requirements and task breakdowns, see `PRD.md`.
